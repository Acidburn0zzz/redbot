redbot by mnot

RED checks HTTP resources to see how they use HTTP, makes suggestions, and finds common protocol mistakes.

Use

A public instance of RED is available at redbot.org.

Alternatively, you can download RED to run locally in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/mnot/redbot

Discuss

The redbot-users mailing list is the place for questions, suggestions and problems to be aired.

RED's tweets carry all of the latest news and information.

See also the issues list.
